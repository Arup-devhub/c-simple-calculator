# Simple Menu-Driven Calculator in C

A lightweight, console-based arithmetic calculator built using C. The program utilizes a `switch-case` architecture to process basic arithmetic operations based on user input, including robust validation for zero-divisor cases.

## Features
* **Addition (+)**
* **Subtraction (-)**
* **Multiplication (*)**
* **Division (/)** — *Includes zero-check validation*
* **Modulo (%)** — *Includes zero-check validation*

## How to Run

### Prerequisites
You need a C compiler installed on your system (like `gcc`).

### Compilation
Open your terminal/command prompt, navigate to the project directory, and compile the file:
```bash
gcc switchcase.c -o calculator
